# swtUI4
基于swt，jface特殊场景还会集成awt，swing等组件封装为一体化的java桌面应用程序框架，定制自己的界面美化特效，定制自己的升级策略，当内部集成jvm，还可以提供跨平台无jdk的运行程序，希望有兴趣的可以一起维护和扩展，扩展rcp，swt方向在国内市场的应用。
